# Modules

`@tsonic/nodejs` currently covers a practical Node-style module set, including:

- `fs`
- `path`
- `http`
- `crypto`
- `events`
- `process`
- `stream`
- `net`
- `tls`
- `url`
- `dns`
- `dgram`
- `zlib`
- `querystring`
- `readline`
- `os`
- `perf_hooks`
- `child_process`

The package is not documented here as a byte-for-byte clone of Node. It is a
curated first-party source package for Tsonic.
